Game Instruction:

  • This a simple coin sorting game.
  • It has 3 levels...
    • Easy: User can generate atmost 10 coins.
    • Medium: User can generate atmost 30 coins.
    • Hard: User can generate atmost 90 coins.
    • There is a "Quick Play" option for the players which is able to generate more than 90 coins.
  • Player will have the option of generating any number of coins.
  • Please note that all the coins are connected to each other.
  • The Goal: To flip all the tails to heads.
  • The player has the freedom to start form any direction (middle, start or end).
  • Once the player has flipped all their coins, simply click the Submit button to see if they won.
  • If the player flips all the coins to heads then he/she wins. If not then they lose.
